## 五、使用说明

### 5.1 查看任务

1. 登录后默认进入工作台
2. 在"任务管理"区域切换不同 Tab：
   - **我的任务**: 查看分配给我的任务
   - **我指派的任务**: 查看我指派给他人的任务（仅 leader 及以上）
   - **待跟进任务**: 查看商机相关的待跟进任务
   - **商机任务**: 查看我负责的商机

### 5.2 创建任务

1. 点击"我的任务" Tab 右上角的"新建任务"按钮
2. 填写任务信息：
   - 任务标题（必填）
   - 任务描述
   - 日期（必填）
   - 优先级（高/中/低）
   - 指派给（leader 及以上可指派给他人）
3. 点击"确定"保存

### 5.3 管理任务

**快速操作**:
- 点击"开始"按钮：将任务状态改为"进行中"
- 点击"完成"按钮：将任务状态改为"已完成"
- 点击"编辑"图标：修改任务信息
- 点击"删除"图标：删除任务（需确认）

**批量查看**:
- 任务按日期和优先级排序
- 逾期任务用红色标记
- 今天的任务用橙色标记

### 5.4 配置菜单权限

1. 以 admin 身份登录
2. 进入"系统管理" > "菜单权限管理"
3. 选择要配置的用户
4. 勾选该用户可以访问的菜单项
5. 点击"保存权限"

### 6.3 数据过滤

**我的任务过滤逻辑**:
```javascript
const filteredMyTasks = myTasksData.filter(t => {
  const taskDate = dayjs(t.date);
  const diff = taskDate.diff(dayjs(), 'day');
  return diff >= 0 && diff <= 7 && t.status !== 'done';
});
```
