## 🌟 系统特色

### 1. 角色权限体系
- **admin**: 超级管理员，所有权限
- **sales_director**: 商务总监，查看辖区所有数据
- **leader**: 组长，查看本组数据，审批申请
- **member**: 普通成员，查看/编辑自己的数据

### 2. 数据隔离
- **member**: 看自己
- **leader**: 看本组
- **sales_director**: 看辖区
- **admin**: 看全部

### 3. 业务流转可追溯
```
线索 → 策略 → 研发任务
```
全链路打通，数据关联清晰

### 4. 实时通知机制
- 任务分配自动通知
- 状态变更自动通知
- 支持通知中心查看历史

### 5. 统计分析
- 各模块提供数据统计
- 可视化图表展示
- 支持多维度筛选

## 🛠️ 技术说明

### 技术栈
- **前端**: React 18 + Ant Design 5
- **后端**: Express.js + better-sqlite3（本地 SQLite）
- **数据存储**: server/data.db

### 启动方式

**开发环境**:
```bash
# 同时启动前后端
npm run dev

# 或分别启动
node server/index.js        # 后端 (端口 3001)
cd client && npm start      # 前端 (端口 3000)
```

**生产环境**:
```bash
# 方式1: 直接运行
NODE_ENV=production node server/index.js

# 方式2: Docker 部署
docker build -t relation-manager .
docker run -d -p 3001:3001 \
  -v $(pwd)/server/data.db:/app/server/data.db \
  --restart=always \
  --name relation-manager \
  relation-manager
```

### 关闭服务
```bash
lsof -ti:3000,3001 | xargs kill -9
```

### 数据库备份
```bash
# 备份
cp server/data.db server/data.db.backup

# 恢复
cp server/data.db.backup server/data.db
```
