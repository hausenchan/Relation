## 使用建议

### 查看文档

**在 GitHub/GitLab**:
- 直接在仓库中点击 `.md` 文件即可查看
- 支持目录导航和锚点跳转

**在本地**:
- 使用 VS Code、Typora、MacDown 等编辑器
- 使用 `grip` 命令行工具预览：
  ```bash
  pip install grip
  grip 进度.md
  ```

**在浏览器**:
- 安装 Markdown 预览插件
- 或使用在线 Markdown 编辑器

### 更新文档

**进度更新**:
1. 打开 `进度.md`
2. 在对应的 Phase 下添加新功能
3. 更新"最后更新"日期

**功能说明更新**:
1. 打开 `功能说明.md`
2. 在对应章节下添加或修改功能说明
3. 更新"最后更新"日期
