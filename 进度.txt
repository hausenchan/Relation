🗒️ 技术说明
-----------
- 前端：React 18 + Ant Design 5
- 后端：Express.js + better-sqlite3（本地 SQLite）
- 数据存储：server/data.db
- 启动后端：node server/index.js（项目根目录）
- 启动前端：cd client && npm start
- 同时启动：npm run dev（根目录）
- 关闭服务：lsof -ti:3000,3001 | xargs kill -9
- Docker 部署：docker build -t relation-manager . && docker run -d -p 3001:3001 -v $(pwd)/server/data.db:/app/server/data.db --restart=always --name relation-manager relation-manager
